# 🛒 Hailow — Marketplace Platform

A microservices-based marketplace platform where sellers can create storefronts, manage products, and buyers can browse catalogs, manage carts, and place orders. Built with **Go** and **C#** using **gRPC**, **RabbitMQ**, **PostgreSQL**, **Redis**, and **S3-Storage**.

## Clone

```bash
git clone https://github.com/Blackflame796/Hailow.git --recursive
```
